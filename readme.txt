A Pocket importer to use with the Keyring and Keyring Social Importers WordPress plugins.

== Description ==

Own the data that you create. Install the Keyring and Keyring Social Importers plugins, and then use Keyring Pocket importer to save copies of the links you've shared on getpocket.com.

After installing, visit Import > Pocket to get started.

When you set up your Pocket App connection, your app will need to have "retrieve" permissions.
